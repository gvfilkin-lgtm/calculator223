# Калькулятор сроков по 223-ФЗ

Современный калькулятор сроков проведения закупок по Федеральному закону №223-ФЗ.

## Цель проекта

Создать современный аналог калькулятора сроков закупок с полностью собственной архитектурой, современным интерфейсом и возможностью дальнейшего изменения алгоритмов расчёта.

## Основные возможности

- Все способы закупок по 223-ФЗ
- Пользовательский способ закупки ("Другой")
- Учёт выходных дней
- Учёт официальных праздничных дней РФ
- Учёт переносов выходных
- Автоматический перенос сроков на следующий рабочий день
- Генерация ссылки на расчёт
- Работа как отдельной страницы
- Работа внутри iframe
- Адаптивный интерфейс
- PHP + JavaScript без базы данных
